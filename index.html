Free Runner Demo

Instructions

  • Tap the screen or Press spacebar to jump.
  • Hold for a bigger jump.
  • Tap the top-right or press P to pause.
  • For moblie, landscape orientation is recommended.

This is a project made for the Web Development For Everybody capstone.

Game was made using Godot , A free and open source game engine.
The website it's self is programed using VScode.

Pixel art from Rotting Pixels
